# Scripts Auxiliares y Utilidades

Esta carpeta contiene herramientas de soporte diseñadas para tareas operativas rápidas, diagnósticos y mantenimiento del sistema que no requieren la ejecución de un Playbook completo de Ansible.

### Contenidos
- **Health-Checks:** Scripts rápidos para verificar la conectividad de red y el estado de los servicios (ICMP/HTTP) tras el despliegue.
- **Mantenimiento:** Utilidades para la limpieza de logs, reinicio de servicios específicos o gestión rápida de la base de datos MariaDB.
- **Troubleshooting:** Comandos de diagnóstico para identificar fallos en la comunicación entre el nodo de control y los nodos de Proxmox.

### Objetivo
Ofrecer una vía táctica y ágil para la resolución de problemas (Troubleshooting) y el mantenimiento diario de la infraestructura, complementando la automatización estratégica realizada con Ansible.
