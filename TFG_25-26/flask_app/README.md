# Interfaz de Gestión y Visualización (Flask)

Este directorio contiene el código fuente de la aplicación web diseñada para la monitorización e inventariado del laboratorio.

### Contenidos
- **app.py:** Punto de entrada de la aplicación y lógica de rutas.
- **templates/:** Archivos HTML con la estructura de la interfaz de usuario.
- **static/:** Recursos estáticos como hojas de estilo (CSS) y scripts del lado del cliente.
- **models/:** Definición de la interacción con la base de datos MariaDB.

### Objetivo
Proporcionar una capa de visualización amigable que consuma los datos de la infraestructura y permita consultar el inventario de servicios de forma centralizada.
